## 💡 QUESTIONS FRÉQUENTES

**new vs malloc?**
> "new appelle le constructeur."

**delete vs free?**
> "delete appelle le destructeur."

**Pointeur vs référence?**
> "Pointeur peut être NULL, référence toujours valide."

**OCF?**
> "4 fonctions: constructeur, copy, assignment, destructeur."

**Polymorphisme?**
> "virtual permet d'appeler la bonne fonction selon le type réel."

**Destructeur virtuel?**
> "Obligatoire pour delete sur pointeur de base."

**Deep copy?**
> "Copier le contenu pointé, pas le pointeur."
